## Interactive Terminal

The portfolio features an integrated **Neural-Core Terminal**. Visitors can interact directly with the system using various commands.

**Available Commands:**
| Command | Action |
| :--- | :--- |
| `help` | Displays all available system commands. |
| `software` | Lists the current software lineup (sweep-it, arch-it, etc.). |
| `matrix` | Activates the visual override mode. |
| `clear` | Purges the terminal history. |
| `ping` | Tests the latency to the Neural-Link. |
